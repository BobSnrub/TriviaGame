How to Play:

This is a simple trivia game where you are given a question and you answer it by clicking on one of the four multiple choice buttons. There are 20 questions in each quiz and each difficulty is set on a timer where you have a certain amount of time to answer the question before it is counted as incorrect.

Rules for Hard:

For hard you have 5 seconds per question to answer.

For Medium:

For medium you have 15 seconds per question to answer.

For Easy:

For easy you have 30 seconds per question to answer.

Bonus:

You get one incorrect answer removed from your score if you correctly answer five questions in a row.
